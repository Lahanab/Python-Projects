# Python Programming Projects

## Overview
This repository contains a collection of Python projects designed to demonstrate core programming fundamentals, logical thinking, and problem-solving skills. The project's focus is on writing clean, readable code and applying Python concepts through hands-on applications.

These projects reflect practical learning and are suitable for showcasing foundational Python skills for entry-level technical and data-related roles.

## Tools & Technologies
- Python
- Standard Python Libraries
- GitHub

## Repository Contents

### Projects

- **Expense_Tracker**  
  A Python application that tracks expenses, categorizes spending, and helps analyze personal financial habits.

- **Guess the Number**  
  A number guessing game that uses loops, conditionals, and user input to reinforce control flow concepts.

- **Mad Libs**  
  A text-based program that demonstrates string manipulation and user interaction.

- **Rock, Paper, Scissors**  
  A classic game implementation showcasing conditional logic, randomization, and input validation.

- **Tic_Tac_Toe**  
  A console-based game that applies functions, loops, and game logic to manage player turns and determine outcomes.

- **YouTube**  
  A Python project focused on automating or interacting with YouTube-related tasks, demonstrating scripting and basic automation concepts.

## Key Skills Demonstrated
- Python syntax and programming fundamentals
- Control flow (loops and conditionals)
- Functions and modular code design
- User input handling and validation
- Logical problem-solving and debugging
